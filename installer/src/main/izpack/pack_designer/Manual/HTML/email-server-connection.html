Email server connection


Topic content

Description

Environment entries of type Email server connection are used to define an access point to an email server for email exchange.They define a connection between Orchestra and an email server.

Purpose of object

It is used by the channels Email Listener and Email Sender to send and receive emails to the server specified in the entry.

Configuration

View of environment entry email server connection

Step by Step

1.Create an Environment entry of type Email server connection.

2.Fill in the host or IP of the email server. E. g. for email address xyz@soffico.de the host is soffico.de

3.Optionally a trusted server credential can be selected. Therefore create Credentials of type trusted server and add the intended keypair.

4.Select the protocol to use for fetching emails:

oIMAP/IMAP Secure: Stores emails on email server

oPOP3 /POP3 Secure: Stores emails at local system

oEWS 2007-20010: Web service to access  mailbox on exchange server

5.The port will be filled in automatically, but can be changed.

6.The default folder to fetch is commonly INBOX. If it’s necessary to choose another one please ask your email administrator.

7.Select the protocol to use for sending emails: SMTP/SMTP Secure: Simple protocol for sending emails

8.The port will be filled in automatically, but can be changed.

9.Click Test: An alert window will appear, which shows whether the host is reachable.

10.If the test was successful click Finish.
