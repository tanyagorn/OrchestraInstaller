Orchestra Parameter Receiver


Topic content

Description

Orchestra Parameter Receiver is used to create a connection between two Orchestra nodes. It is build up like signals, but with a point-to-point connection.  It is used to transfer data between distributed orchestra instances.

The inbound channel implements a "idempotent" behavior. This means, the receiver can determine if the current request is sent repeatedly. When it detects such a repetition the request is skipped and not executed anymore. The sender receives an "already-executed" result. The sender can define if idempotent communication should be used or not. When the channel is invoked by a process activity, just set the parameter "IDEMPOTENT" to false.

Step by Step

•Authentication and Authorization - HTTP authentication: basic: user/password digest: user/password certificate authentication: authentication of the client via certificate (X509).
Authorization means that the inbound channel checks if the user authenticating itself has a certain role assigned.

•Mode - Defines the mode of operation. When set to request, orchestra submits the request, but doesn't wait for completion. Otherwise, when synchronous is used, the execution blocks until a result is received.

•Timeout - After this time a HTTP 205 (Reset Content) is returned to the sender. The Orchestra Parameter Sender reacts with an error ("A timeout occurred during the execution") to this code.

•Parameters - Define one or more parameters that that must be send from the remote orchestra

•the mapping of the parameters between outbound and inbound channel is done by name (order of the parameters is not important)

•the parameters must be set manually at outbound and inbound side (there is no export/import)

•the usage setting (INPUT, OUTPUT) is only used in request/response mode.

 

Testing

In the Scenario with the Inbound channel start the test engine, open the tab Channels, open the Properties of the Parameter Receiver and from there copy the URL. With this URL you man then configure the Parameter Sender.
