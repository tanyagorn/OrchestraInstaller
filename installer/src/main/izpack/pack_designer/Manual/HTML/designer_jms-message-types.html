JMS message types


Topic content

Definition

The channel generates different inbound messages per JMS message type

Parameters

Parameter

Description

TextMessage

Relative to the used deserializer, an Orchestra message is created

BytesMessage

Relative to the used deserializer, an Orchestra message is created

StreamMessage

An Orchestra message is created, this has the input stream in a <value> field

MapMessage

An Orchestra message with all key value pairs is created

ObjectMessage

An Orchestra message without any content is created, a input object with type "any" is created which holds the input object.
