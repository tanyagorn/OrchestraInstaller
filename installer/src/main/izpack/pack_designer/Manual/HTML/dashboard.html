Dashboard


Topic content

The dashboard aggregates essential information on current scenarios, as well as the server itself. It shows at a glance the full system capacity, the CPU usage, the general memory usage and the memory in use by the Java runtime environment.

Technical Monitoring – Dashboard

 

In addition, the start time, operation time and the Orchestra version is displayed.

Technical Monitoring – Runtime

 

There is an overall view for observing the calls to or from external systems, have a look at info box "Channel invocations". For each channel type you get informed about quantity of usage and the successful resp. failed calls. Look like Figure a, you can also see your signal communication at a glance.
