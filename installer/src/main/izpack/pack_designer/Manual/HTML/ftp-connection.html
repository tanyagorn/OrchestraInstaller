FTP connection


Topic content

Description

Environment entries of type FTP connection define the parameters to access an FTP server. They define a connection between Orchestra and a Server using a FTP protocol.

Purpose of object

It is used by the channels FTP Listener and FTP channel to declare the server to access.

Configuration

Base view of environment entry FTP connection

Step by Step

1.Create an Environment entry of type FTP connection.

2.Select a Protocol to use for connecting:

oFTP: Unencrypted protocol for data transfer. Uses separate TCP connections for commands and data.

oSFTP: Protocol enables encryption, it exchanges commands using a SSH (Secure Shell) connection. A lot of parallel logical connections can run through one TCP channel.

oFTPS: Encryption protocol, uses FTP routing its commands and data through a secure connection using SSL (Secure Sockets Layer) or TLS (Transport Layer Security). Like FTP it uses separate TCP connections for commands and data.

3.Select Authentication type (SFTP and FTPS)

4.Select Security Protocol (FTPS)

5.Select Security Mode:

oexplicit: Client has to explicitly request security from server. Server can allow or refuse insecure connection.

oimplicit: Encryption by SSL/TLS is required from server.

6.Select whether to use active mode:

oActive mode: Server opens a data connection to the client

oPassive mode: Client opens a data connection to the  server

The original FTP specification defined Active mode. But with the introduction of firewalls this caused problems because in this case the server has to connect to a random client port which nowadays in most cases is blocked by the firewall. Therefore today the Passive mode is used almost everywhere.

7.Enter Host or IP of the FTP server e.g. ftp.google.com or 127.0.0.1.

8.The command Port is usually 21 but can be changed.

9.Optionally enter Initial Directory in servers file system.

10. Click Test: An alert window will appear, which shows whether the test of the connection was successful.

11. If the test was successful click Finish.
