Worklist Credentials


Topic content

The work list credentials page has two tabs. Selecting the "User" Tab opens a page that  is nearly the same as the page for the monitor users from the topics Create a user and Edit User - assign access rights. So use these topics if you have questions here. The only difference is that not all options are available for work list users.

The tab "Groups" offers a table of all existing work list user groups, like the following picture shows.

Page for group management

If you need a new group of users, click on the create new group button and enter name and description of the group. It is now available in the "Groups" section of the user edit dialog. The table has also a context menu for deleting and editing the groups.
