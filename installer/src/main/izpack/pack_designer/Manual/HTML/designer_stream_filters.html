Stream filters


Topic content

Concept

A stream filter is used to change byte streams while they are written by a Serializer or read by a Deserializer.

Creation

A stream filter is always created int the context of configuring a Stream channel. Below the line for selecting the serializer you will find a line where you can add and select a stream filter by double clicking the arrow --> This way you also may build a sequence of filters.

several stream filters forming a chain

To configure a filter the user has to double click on the name in the sequence.

To remove a filter select it and click the right mouse button. In the popup menu then select the entry Delete.
