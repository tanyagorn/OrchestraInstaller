Partner


Topic content

 

1.Open the contextual menu of Partner by right-clicking and choose New Partner from Partner Type.

2.Name the Partner and choose the Partner Type.

 

3.Click on new to create and name a new Subscription.

 

At this Point you are able to choose the Business Objects which the Partner should receive.

 

4.By clicking on Edit Mapping you are able to change the configurations like you can see in the graphics below.

 

5.In order to identify the Partner you have to put in its name.

 

6.By blacklisting items you can select Business Objects which should not be received by the Partner

7.With the options on the Named Mapping tab you can change the file-name under certain conditions

 

8.On the Technical Settings tab you are able to select the path to save the received objects. Select test to check if the selected path exists.

 

9.Save your data and open the contextual menu of the created Partner. After that select deploy and choose a path where you want to store the file.

Don't forget to set up the Business Objects in the sending scenario like it is shown in the next images.
