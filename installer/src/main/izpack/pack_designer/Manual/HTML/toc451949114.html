Manual Creating


Topic content

In order to manually create REST Service Client following steps need to be performed.

1.Right Click on the REST Client Service option from orchestra designer main menu.

2.Select Create option to create a new REST Client Service entry.

Figure 45 – REST Service Client - create option

3.In the opened dialog enter the Name and Description (optional) for your REST client and click on the Finish button.

Figure 46 – REST Service Client - create dialog

4.In the following, the configuration panel for the created REST service client will be shown.

5.Setup URL value by selecting a URL environment variable. For this create an environement entry with type URI.

Figure 47 – REST Service Client - configuration panel

6.After applying configuration setting to the Resources element you will need to configure each Resource element with corresponding Methods.
