GUI


Topic content

To see the state of the Juno instance open a browser and type url: http://localhost:8080/Juno.html. If you changed the port of Juno set your port instead of 8080.

You will get to the overview screen of this Juno. The overview is just for observing purposes. You are not able to change, start or stop anything.

The first part of the view are the connection details of the  actual Juno instance:

 

Information of this communicator instance

 

The second tab contains information about the connected orchestra:

Information of connected orchestra node

Afterward an overview of the actual running channels is visible. The view is equally designed as the channel view in orchestra monitor.

Overview of channels, running on this communicator instance.

The table displays only running channels. If the scenario is not started, the channels won't be visible in this view.

 

Additionally you are able to open a detail view for each channel, similar to orchestra monitor by double clicking on the intended channel in table.

Channel detail dialog

 

The last part of the view is a overview of all processed messages. It shows if a message is either send or received, processed asynchronous or synchronous and if it is a request or a response. Additionally a little description is given, why this message is send. The view actualizes automatically and is sorted by newest message:
