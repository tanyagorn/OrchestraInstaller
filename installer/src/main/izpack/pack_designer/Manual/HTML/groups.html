Groups


Topic content

Description

Group is a helper concept that enables grouping of roles in one scope. E.g. Admin group might refer to create, read, update and delete roles. Afterward, when configuring Credentials you can simply assign the Group to it, instead of adding all mentioned Roles.

Creation

Please navigate to Security->Groups in the scenario elements tree. Right click on it and select Create option:

Creating a group from context menu

The creation dialog will be opened where basic information has to be set:

Creating new group

Configuration

The configuration allows Description change, as well as Role assignment:

Configuring a group

When you want to assign specific role to the group, simply select check box on the left side.
