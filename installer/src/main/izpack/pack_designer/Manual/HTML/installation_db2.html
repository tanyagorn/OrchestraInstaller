Configuration Adaptions

The needed SQL Scripts for the Orchestra Installation can be found on the Orchestra CD in ...\Database\db2

1.Start the DB2 command editor

2.Open the script 00_sys_tablespace.sql with it.

3.Change the paths for the REGULAR TABLESPACE in the script (e.g. C:\DB2\NODE0000\SQL00001\*)

4.Run the script as User SYSTEM.

5.Open the script 01_sys_create_schema.sql.

6.Run the script as User ORC.

7.Open and run all other scripts 02-XX as User Orchestra.

Deadlocks can occur if DB2 must handle big amounts of data. To avoid that problem, run the following commands in the command editor.

db2set DB2_RR_TO_RS=ON
db2set DB2_EVALUNCOMMITTED=ON
db2set DB2_SKIPINSERTED=ON
db2set DB2_SKIPDELETED=ON
