Deployment of a scenario


Topic content

Deployment is the process of transferring a scenario from design to runtime status. The .psc file of the scenario is deployed in the runtime by uploading them.

To deploy a new scenario, open the web interface and select subitem Scenario below Administration on the navigation bar. As you see on the screenshot below, click the button "Deploy new scenario" first to open the upload form. This dialog enables you to select your .psc-file from local file system and submit it to the server for deployment.

Steps to deploy a scenario

 

In this case the scenario is checked and uploaded to the server. The scenario remains disabled and has to be started by a user manually.

Note:

•Before starting up the interface, please verify the landscape settings.

•After selecting a scenario file, you can assign groups to the new scenario, by checking the  specific check box.
