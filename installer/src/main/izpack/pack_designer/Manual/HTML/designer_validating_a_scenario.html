Validating a scenario


Topic content

In order to avoid problems with the scenarios you are building there is a possibility to validate your single elements or all at the same time.

 

Clicking on the Validate all is going to validate all the elements used in your scenario. If there are errors occurring while validating they are shown up at the problems tab like it is shown within the next picture.

By double-clicking the configuration of the specific element appears. Together with the warning or failure notice you should be able to solve the problem.
