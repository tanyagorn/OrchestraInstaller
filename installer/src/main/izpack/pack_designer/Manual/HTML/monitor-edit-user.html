Edit User - assign access rights


Topic content

The user details dialog contains next to [General] and [Authentications] three more tabs.

Figure 4‑20 User permissions

 

Orchestra distinguishes between group, roles and scenario permissions. The configuration of these permissions can be done by using the remaining tabs on the user details dialog (Groups, Roles and Scenarios).

A permission group is a collection of multiple roles. If a group is enabled, you can not disable the group-related roles manually at the Roles tab.

ATTENTION! Be aware the permission management is a very powerful tool and it should be used by experienced Orchestra system administrators only. In Order to allow access to the monitor at any time, please do not adapt the roles and groups of Orchestra default users (admin and monitor).

 

For more detailed information about the permissions and their meaning please check the tables below.
