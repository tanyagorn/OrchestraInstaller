Adapt the landscape settings


Topic content

The settings below subitem Landscape serves to maintain the technical infrastructure. At this point, the specific environment settings of a scenario can be configured. (e.g. database connection or file path)

Figure 3‑16 Edit and test environment entries

 

To do this, select the desired scenario and double-click to the appropriate environment entry. Use the appearing form to adjust the contained parameters. At this point a test is available for many Environment Entries.

 

When the user confirms the changes with save, the documentation of the scenario is automatically updated. This is done asynchronously in a background task
