Alert System


Topic content

Running a communication server like Orchestra requires it to work in a durable and stable manner. In case of errors it may be necessary that an administrator can recognize critical states to be able to take specific action. Therefore Orchestra must be able to give reasonable and timely notice about critical states.

The alert system allows to define rules valid for a group of scenario elements. It is possible to specify notification actions to be executed when one of the rules is not valid.
