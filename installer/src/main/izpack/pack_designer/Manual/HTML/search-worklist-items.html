Search worklist items


Topic content

Description

Search DICOM worklist items in a DICOM worklist.

 

Purpose of object

This scriptlet is used to search within a DICOM worklist for DICOM messages having certain properties.

To be more flexible within the script the search keys are configurable.

By default the "PatientID" is used as search key.

 

Create

In the process model editor open the panel named process, open the tree node DICOM worklist, select the node Search worklist items and click into the process model to create a new invocation of this scriptlet.

Icon

 

Configuration

Properties and Parameter assignment

The assignment sections define the assignment of variables between the script and the process model.

The following parameters are available:

Ingoing parameters:

•worklistName the name of the worklist where to search DICOM messages.

•PatientID the ID of the Patient to search for.

 

Outgoing parameters:

•ITEMS messagelist containing the found DICOM messages.
