MLLP Sender


Topic content

Description

This channel is used to send HL7 messages over Mini Lower Level Protocol.

Configuration

The dialog to configure the MLLP Sender looks like:

Mode: Choose synchronous or asynchronous.

Connection mode: Choose if the connection shall be temporary or durable

LLP Handshake (ACK): Choose whether an acknowledgement shall not be send (MLLP version 1), or shall be send automatically (MLLP version 2)

MLLP-Framecharacters: Define the TCP connetion to the receiver (Hostname /IP and port).

Synchronous timeout: This is the time, this channel will wait for the process to finish.

Serialization: Choose a serializer for the request message. (By clicking the -->-arrow, a stream filter can be defined)

De-Serialization: Choose a deserializer for the response message. (By clicking the -->-arrow, a stream filter can be defined)
