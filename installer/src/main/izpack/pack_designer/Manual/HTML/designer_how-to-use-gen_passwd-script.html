How to use gen_passwd script


Topic content

For generating encrypted passwords for Orchestra, the script gen_passwd.cmd for Windows respectively gen_passwd.sh for Linux are provided.

That the script is runnable, the file orchestra_env.bat (respectively orchestra_env.sh) have to be adapted appropriately.

 

For creating a new encrypted password, you have to hand over the character sequence that should be encrypted to the script:

gen_passwd.sh <plaintext_password>

If a encrypted character sequence should be decrypted, you have to pass "decrypt" as a second parameter.

gen_passwd.sh <encrypted_password> decrypt
