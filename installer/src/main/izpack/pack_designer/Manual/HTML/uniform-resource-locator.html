Uniform resource locator


Topic content

Description

Environment entries of Uniform resource locator are used to identify and localize resources. It can be possible, that a address information in test system differs from address information in target system.

Purpose of object

URL´s can be used in HTTP Oubound channels: HTTP Caller, HTTP Post Sender or HTTP Sender (Parameter). In this case the URL defines the  website to communicate with.

Configuration

Base view of environment entry uniform resource locator(URL)

Step by Step

1.Create an Environment entry of type TCP/IP Uniform resource locator.

2.Fill in the URL: e.g. http://soffico.de or file:///C:/path/file.txt.

3. Choose whether to use a Proxy server, which sits between you and the intended address. Therefore generate an environment entry of type Proxy server.

4.Click Test: An alert window will appear, which shows whether the test of the connection was successful.

5.If the test was successful. Click on Finish.
