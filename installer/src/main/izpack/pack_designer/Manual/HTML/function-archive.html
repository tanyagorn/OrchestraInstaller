Function archive


Topic content

Description

oA function archive is a standard java archive that contains user defined mapping functions

oWhen the archive is added to the scenario, the mapping functions are available in the context menu of the graphical mapping

oA user can use the functions in the same way as any other mapping function

 

Creating a custom function

o Create a java project (e.g. in eclipse that) that will be used as development environment

oEnsure that the following libraries are added to the classpath of your project

▪orchestra-domain-<version>.jar

oThis file contains the necessary definitions

 

oNow create your java functions as static member functions of an arbitrary java class.

oNote: In order that orchestra recognizes a function as a mapping function you have to add the annotation "@OrchestraFunction"

 

oExample:

package soffico.test.pckg;

 

import emds.mapping.OrchestraFunction;

 

public class UserDefinedMappingFunctions {

 

  @OrchestraFunction(description="Description of the mapping function",author="Author",date="Creation date" )

  public static String concatWithSpace( String a, String b ) {

           return a + " " + b;

  }

}
