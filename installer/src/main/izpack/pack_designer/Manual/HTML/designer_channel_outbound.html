Outbound


Topic content

Concept

An outbound channel is a Channel which is invoked by another Scenario Element.

Purpose

Typically an outbound channel is invoked from a process (an instance of a Process model) if the process wants to send data to an external entity, like a file system, a database or another program or to fetch data from an external system. The notion Outbound in this context means only that the call is initiated from the process model, it does not denote the direction into which data are actually transferred.
