Creating a scenario library


Topic content

A scenario library is created like a normal scenario, except the type of the scenario. It must be set to "scenario library". This step is shown in the following figure.

Creating a scenario library

 

After the creation of the scenario all functions can be used that are known from the standard scenario development.

 

Note: All scenario libraries must be stored same library directory. This setting can be done in the Preferences-Dialog of the Orchestra Designer.
