Delete worklist items


Topic content

Description

Delete DICOM worklist items with given AccessionNumber.

 

Purpose of object

This scriptlet is used to delete ITEMS in a DICOM worklist which actually is a named message list.

To be more flexible, within the script the keys identifying the message or the messages to delete are configurable.

By default the "AccessionNumber" is used to identify the message which is to be deleted.

 

Create

In the process model editor open the panel named process, open the tree node DICOM worklist, select the node Delete worklist items and click into the process model to create a new invocation of this scriptlet.

Icon

 

Configuration

Properties and Parameter assignment

The assignment sections define the assignment of variables between the script and the process model.

The following parameters are available:

Ingoing parameters:

•worklistName the name of the worklist where to put the DICOM message.

•accessionNumber identifier of the message to delete.

 

Outgoing parameters:

• deleteCount number of deleted items.
