Motivation


Topic content

Orchestra supports companies with the implementation of a strategically planned integration architecture, which is in contrast to an integration landscape that has grown historically. Within the integration architecture, Orchestra acts as central platform, which provides and operates all data communication methods and procedures.

 

The number of interfaces can be significantly brought down. With the introduction of a new system, there is no need to develop a distinct interface for each system (as with conventional point-to-point connections). Instead, the mapping to the meta-format of the Orchestra ESB must be only performed once.

 

Central, cross-system flow control in Orchestra is in contrast to more or less user-specific flow scripts (hard-coded).

 

The Orchestra ESB is a modular, but highly integrated platform for the connection of all existing applications within the company. The Orchestra architecture helps companies to constantly integrate new applications or replace them. The support of publish/subscribe mechanisms for data communication decouples applications more powerful than conventional point-to-point connections.

 

The modular composition particularly supports the integration of heterogeneous system architectures, which require the integration of different vendors and technologies. For this purpose, soffico provides a multitude of ready-for-use adapters. In addition, for the integration of in-house developments, Orchestra supports the development of your own adapters.  

 

The scalability of the Orchestra architecture allows for the operation of Orchestra in almost all areas and orders of magnitude - ranging from the small, medium-sized company to the global corporation that processes 100 million messages every day. The size of the Orchestra installation can be individually adjusted to current requirements, and can then evolve in line with changing requirements of the management.  

 

Orchestra supports the approach of a largely code-free interface development. Thereby, development is accelerated, initial training of employees regarding foreign interfaces is facilitated, and thus overall maintainability is increased.  

 

Orchestra supports the service-oriented architecture (SOA) paradigm. All components of the Orchestra Suite are designed to support the structure, management and control of appropriate architectures in organizations.
