Import WADL file


Topic content

Orchestra provides you with an ability to create REST Service Client object by importing WADL file definition.

1.Click on the File > Import > REST service declaration option from the designer top menu.

Figure 48 – REST Service Client - import WADL file

 

2.You will be presented with the Import Rest service description dialog.

3.Select the WADL file from your directory and click the Next button.

4.On the second screen select the Create service client option and click on the Finish button.

                         

Figure 49 – REST Service Client - import Rest service wizard

 

5.After clicking the Finish button, new REST Service Client object will be created and the configuration panel will be opened.
