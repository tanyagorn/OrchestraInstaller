Excel reader


Topic content

Description

The excel reader parses excel files and creates a structured Message from it.

Creation

A excel reader is created, like all other deserializers, by selecting it in the according Stream channel.

Configuration

The user can configure the excel reader using the following dialog.

 

Sheet: Insert the used sheet

Skip lines: Lines that shall be skipped (e.g. if there is a title)

Ignore empty cells: Check to ignore empty cells

Ignore empty lines:  Check to ignore empty lines

Typed: If checked, Orchestra will look up the type of the cell
