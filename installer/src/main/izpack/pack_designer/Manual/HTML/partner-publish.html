Partner-publish


Topic content

Description

The feature is only usable in the context of partner management. It throws a message and the name of a business object to the ESB triggers a process model for any partner that has subscribed to the business object.

Create

1. Create a process model

2.Then point the mouse on "process", as default, this window is on the right side of the Orchestra window.

3.Click on the folder Enterprise Feature to see its content.

4.Click on "partner-publish".

5.Click on any place in your process model.

6.Integrate it in process schedule, by clicking on the service-response icon and drag and drop arrows to the to be connected item.

7.Connect it by clicking on the partner-publish icon and drag and drop the arrow to the intended service. A dashed arrow should appear.
