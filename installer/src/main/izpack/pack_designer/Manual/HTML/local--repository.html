Create a local repository


Topic content

Just choose File-->Versioning-->Share scenario

Like in the Chapter on Share a scenario  a window opens where you can choose the repository type.

Choose Git and on the next window select the checkbox local versioning only.

 

 

Then click okay and you are done.  You have created your first local git repository.

Now you can do anything previously described in the versioning section.

In the section How to share a local git repository with others you will learn how to share  your local repository in a remote repository.
