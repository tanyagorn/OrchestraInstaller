Bindings


Topic content

Description

Bindings are used to group one or more operations in specific communication protocol and message data format. Also, one service can have more bindings and in each of them you can define how the service and its operations are used.

Creation

For creating a binding you need to navigate to the ServiceDeclaration->Bindings in service declaration panel (result from service declaration creating):

After selecting Add binding option you will be asked for binding type:

Binding types

Orchestra defines following binding types that are explained in separate chapters:

•SoapBinding

•OrchestraService

•NotSupportedBinding

•JavaApiBinding

Configuration

Configuration is specific for binding type, please refer to binding chapters for details.
