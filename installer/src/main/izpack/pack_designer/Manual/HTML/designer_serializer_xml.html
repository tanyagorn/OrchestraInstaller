XML writer


Topic content

Description

The XML writer is a serializer that transforms a structured Message (internally represented as DOM tree) into an XML data stream. In Orchestra every structured message can be written as XML.

Creation

A XML writer is created, like all other serializers, by selecting it in the according Stream channel.

Configuration

The user can configure the XML writer using the following dialog.

Configuration for the XML serializer

•Encoding Here you select the character set and encoding of the text stream

•Content type Here you can change the content type: Possible alternatives are text/xml, text/html, application/xml and application/xhtml+xml

•Document type declaration Rarely used; in this field the user may declare a DTD for the XML header e.g. html to write <!DOCTYPE html>

•Processing Instruction Rarely used; in this field the user may declare a PI for the XML header, typically a stylesheet declaration

•Insert newlines If this option is selected

•Encode non ASCII as entity ref If this option is selected (rarely used)
