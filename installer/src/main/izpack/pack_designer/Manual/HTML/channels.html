Channels


Topic content

Concept

A channel serves to exchange data between Orchestra and external resources like databases, file systems and other programs like Web servers. Another commonly used word for this type of object is adapter.

Creation

A channel is created in the Designer by selecting the group channels in the tree of scenario elements, clicking the right mouse button and selecting the item Create channel.

select pop menu item

 

Then the dialog Create new channel pops up.

1.First you enter an appropriate name for the new channel and optionally a short description.

 

2.After clicking on the Next button you can select the type of channel you want to create

 

There are two types of channels, Inbound and Outbound channels.

3.After clicking the Finish button the channel will be created. A panel is opened where you can configure the new channel.
