Reading a particular appointment by using a Template Parameter (Method GET)


Topic content

 

In this chapter we will create a Orchestra REST Client implementation to request a particular appointment and save it to your File system. According to chapter 3.3.4 Reading a particular appointment by using a Template Parameter (Method GET)  we request the appointmentService by using the URL

http://localhost:8080/AppointmentMgr/appointments/appointments/{id}

1.Copy your shell scenario from chapter 4.5.1 Preparation.

2.Open your process model.

3.Add the Rest Service Client and a call activity to your process model.

4.Link all elements in your modell.

Figure 74 – Reading a particular appointment by using a Template Parameter – Process Modell

5.Now you can configure your call activity for the AppointmentMgr-Service.

As an additional information for telling the server, which appointment slot we want to get, we have to enhance the URL with the Template Paramter id of our Request.

Figure 75 – Reading a particular appointment by using a Template Parameter – REST -Service Method Configuration

 

6.Open the register Service method and select the GET-Method for /appointments/appointments/{id}.

7.Assign the Response MSG of the call to the CURMSG of the process modell.

8.Assign the id-Template Parameter “1”.

Figure 76 – Reading a particular appointment by using a Template Parameter – Call configuration

9.Start your Test-Engine, and have a look at your result-File.
