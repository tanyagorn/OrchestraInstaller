Proxy server


Topic content

Description

Proxy Server Environment entries are used to declare the parameters to access a proxy server. A proxy server is a facilitator between the client and a target system.

Purpose of object

It is can be used for all kind of channels, which access a server. Usually it is used for  a HTTP connection, via channels HTTP GET Receiver, HTTP POST Receiver (General), HTTP POST Receiver (Parameter) ,HTTP Caller, HTTP Post Sender or HTTP Sender (Parameter).

Configuration

Base view of environment entry proxy server

Step by Step

1.Create an Environment entry of type proxy server.

2.Fill in the host or IP of the proxy server. E. g. proxy.example.com or 193.123.456.789.

3.Fill in the intended port, usually 80 or 443.

4.Enter User and Password of the server if necessary.

5.Select Type:

oHTTP: Proxy server only forward HTTP requests(HTTP protocol is necessary). They are used for web access.

oSOCKS: SOCKS proxy server forward every kind of server requests. The SOCKS protocol operates on a lower level than HTTP proxy, therefore it is able to handle also other kind of protocols.

6.A test of proxy server isn´t possible. Click on Finish.
