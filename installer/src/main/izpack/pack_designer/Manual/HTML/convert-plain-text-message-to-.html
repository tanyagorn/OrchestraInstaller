Convert plain text message to HTML message


Topic content

Description

Convert binary message containing plain text to message containing HTML.

 

Purpose of object

This scriptlets takes the constant of a binary messages and places this content within an HTML page. The result is returned as binary message.  

 

Note: The translation requires that the content of the source message could be read as string.

 

Create

See how to create items in process model.

Configuration

Properties and Parameter assignment

The assignment sections define the variable connection between the script execution and process model.

The following parameters are required.

 

Ingoing parameters:

•ALTMSG source message.

 

Outgoing parameters:

• MSG result message as HTML type.
