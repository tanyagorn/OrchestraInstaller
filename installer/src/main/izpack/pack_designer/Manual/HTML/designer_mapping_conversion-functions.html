Conversion functions

The following methods can be invoked in all mappings; graphical mappings as well as procedural mappings. They are used to convert values of different data types to values of other types e.g. parsing or formatting numeric values.


Topic content

•String formatInteger(int value, String format) Create a string from an integral number; the format is given according to java.text.DecimalFormat, e.g. "00000".

•String formatDecimal(BigDecimal value, String format) Create a string from a decimal number; the format is given according to java.text.DecimalFormat (with automatically adapted decimal and group separator.

•Integer toInteger(String str) Parse integer string; if the parameter is null return null, if the parameter is the empty string or no valid integral number then return 0.

•Long toLong(String str) Parse long integer string, skips leading space; if the parameter is null, the empty string or is no valid integral number then return null.

•BigInteger toBigInteger(String str) Parse big integer string, skips leading space; if the parameter is null, the empty string or is no valid integral number then return null.

•BigDecimal toDecimal(String str) Parse decimal string, skips leading space; if the parameter is null, the empty string or represents no valid decimal number then return null.

•Double toDouble(String val)  In the graphical mapping respectively in XPath this function is named number. Parse double string; if the parameter is null, the empty string or represents no valid number then return null

•BigDecimal parseDecimal(String str, String pattern) Convert string to a decimal number; the format is specified as defined in java.text.DecimalFormat (with automatically adapted decimal and group separator)

•Double parseDouble(String str, String pattern) Convert string to a double precision number; the format is specified as defined in java.text.DecimalFormat (with automatically adapted decimal and group separator)

•XMLGregorianCalendar parseDate(String value, String pattern) Create XML date from a string according to a given format, e.g. yyyy.MM.dd means year.month.day (see java.text.SimpleDateFormat)

•XMLGregorianCalendar parseTime(String value, String pattern) Create XML time from a string according to a given format, e.g. HH:mm:ss means hour:minute:seconds (see java.text.SimpleDateFormat)

•XMLGregorianCalendar parseDatetime(String value, String pattern) Create XML dateTime from a string according to a given format, e.g. yyyy.MM.dd-HH:mm:ss means year.month.day-hour:minute:seconds (see java.text.SimpleDateFormat)

•String formatDate (XMLGregorianCalendar cal, String format) Create string from XML date in given format e.g. yyyy.MM.dd means year.month.day (see java.text.SimpleDateFormat)

•String formatDateTime (XMLGregorianCalendar cal, String format) Create string from XML dateTime, xmlDate or xmlTime in given format e.g. yyyy.MM.dd-HH:mm:ss means year.month.day-hour:minute:seconds (see java.text.SimpleDateFormat)

•XMLGregorianCalendar getDate(XMLGregorianCalendar timestamp) Get the date part of a datetime value

•XMLGregorianCalendar getTime(XMLGregorianCalendar timestamp) Get the time part of a datetime value

•XMLGregorianCalendar parseSapDate(String datestr) Create XML date from SAP date string; Normally only used in IDOC mappings

•XMLGregorianCalendar parseSapDatetime(String datestr, String timestr) Create XML dateTime from a SAP date string and a SAP time string; Normally only used in IDOC mappings
