Server Info


Topic content

Helpful information about the server on which Orchestra is running is provided here.

The function "System info" offers the possibility to download a documentation of the server (in case of a distributed system for the selected node).

In a support case all necessary data can be retrieved conveniently in this way and passed on to the Orchestra support.

The archive contains system information like copies of the Orchestra configuration and log files. If the tomcat logging path is set in the environment entries (log.tomcat.path), this directory can be included automatically in the archive.
