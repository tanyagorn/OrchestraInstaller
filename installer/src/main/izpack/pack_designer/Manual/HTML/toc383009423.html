Undeploy a scenario


Topic content

If the scenario is to be completely removed from the system, please use undeployment function.

ATTENTION! This process will delete the scenario together with all belonging Environment Entries and configuration settings permanently.

To do this, navigate to the Scenario menu item to show all deployed scenarios on the server. Prior to undeploy the scenario should be stopped.

There are two ways to undeploy a scenario:

1.Using [Undeploy] button from the context menu available by right-clicking the dedicated scenario

2.open the details view by double clicking the scenario in order to press the [Undeploy] button

For Undeployment you have to confirm your request.
