WebDAV Listener


Topic content

Description

This channel implements a WebDAV server.

Purpose of object

This channel can be used for two different functionalities:

•If a new file is recognized, a trigger event is build

•An external system can have access to the collected messages

Creation

To create a WebDAV Listener follow the steps described in the general description of Channels

Configuration

The dialog to configure the WebDAV Listener looks like:

 

Authentication: HTTP authentication: basic: user/password digest: user/password certificate authentication: authentication of the client via certificate (X509)

Message list settings:

name: Name of the message collection

basepath: Choose how the URL shall be generated: DEFAULT: Path is generated from the scenario ID and channel ID SCENARIONAME: Path is generated from the scenario name COLLECTIONNAME: Path is generated from the collection name

sub directiories: Subfolders of the collection with permissions

directory listing:

HTML-header: structure and appearance of the header of the download site

entry: structure and appearance of the download site

HTML-footer: structure and appearance of the footer of the download site
