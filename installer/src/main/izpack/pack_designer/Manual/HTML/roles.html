Roles


Topic content

Description

Roles are used for authorization purposes that might be required to access Orchestra services from third parties. You can manage different roles based on requirements your system has when it comes to authorization and user management.

Creation

Creating a role is done from the scenario elements tree. Please navigate to Security->Roles and right click on it:

Creating a role from context menu

You will be asked then to provide basic information about the role:

Creating new role

Insert meaningful information about the role and click Finish. The role will be created and is ready for use.

Configuration

Configuring a role allows simply the change of Name and Description.
