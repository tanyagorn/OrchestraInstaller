Logging in


Topic content

Start your browser and enter the following URL: http://<tomcat-host>:<tomcat-port>/orchestra (e.g. http://localhost:8080/orchestra)

The log in window as shown above will be displayed.

The standard installation of Orchestra contains two user with different permissions on the system:

User:                admin
Password:        admin

 

User:                Monitor
Password:        monitor

 

Consider that the log in is case sensitive. The user Monitor has only restricted access rights. The password of the user admin should be changed after the first log in for security reasons.
