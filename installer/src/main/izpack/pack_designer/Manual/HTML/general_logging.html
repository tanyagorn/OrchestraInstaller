Logfiles


Topic content

Orchestra generates its log files in a dedicated directory that was configured during installation process.

As you see in Figure a, viewing log files is not limited to Orchestra. It is possible to specify different logging groups to show log files of other tools like tomcat or project specific logs.

Log File View

 

You can change the log level for Orchestra runtime at the server settings. Supported log levels are given below in descending order.

▪SEVERE, WARNING, INFO, CONFIG, FINE, FINER, FINEST, ALL.
