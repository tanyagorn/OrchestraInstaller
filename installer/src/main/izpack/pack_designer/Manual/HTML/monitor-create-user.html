Create a user


Topic content

To create a new User for Orchestra monitor, open the web interface, log in with an administrator privileges and select subitem User management below Administration on the navigation bar. Select on the dropdown box the item [Runtime] to open the view for web frontend users. Clicking the button [Create new user] opens the user creation dialog. As you can see in figure 4-1 the dialog contains several tabs.

Figure 4‑19 User administration

 

On the first tab [General] you need to enter a user name and if you like a user description. At tab [Authentications] you can set up the authentication method for this user. Orchestra offers password or certificate based authentication.

The certificates client certificate, ssh certificate and X.509 certificate (PKCS#12) are supported by Orchestra.

The most common method in use is password authentication. Instead of setting the password in this dialog you can configure orchestra to authenticate the user against an LDAP directory server.

To set up a authentication enable the checkbox on left side of your desired method. Then click the button [...] on the right side to set the password or upload the certificate. Pressing [Save] finishes the creation process and pressing [Cancel] discard your changes.
