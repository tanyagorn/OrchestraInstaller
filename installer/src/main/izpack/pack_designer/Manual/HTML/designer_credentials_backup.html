Topic content

Concept

Credentials are used to provide identity of a party (either machine or human) in security operations. Based on usage, Orchestra distinguishes three types of Credentials:

- Trusted server

- Client login

- Technical login

Trusted server

Trusted server type of Credential refers to the identity of a machine that Orchestra communicates with. Typical Credential of Trusted server type is Digital Certificate (Public Key Certificate), and it is used in HTTPS protocol - server side authentication.

Client login

Client login is similar to Trusted server, but with difference that it is used to authenticate client side, rather than server side identity. Typical usage is when Orchestra acts as a server side (e.g. by providing a Web Service) and client wants to connect to it over HTTPS.

Technical login

Technical login is actually client authentication of Orchestra to other parties.
