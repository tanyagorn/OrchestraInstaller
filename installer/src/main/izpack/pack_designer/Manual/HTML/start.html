Start


Topic content

Description

Process models need at least one start event. The here described start event just starts the process without any configurations.

 

Purpose of object

The event starts a process.

Create

1. Create a process model

2.Then point the mouse on "process", as default, this field is on the right side of the Orchestra window.

3.Click on the folder start events to see its content.

4.Click on "start".

5.Click on any place in your process model.

6.Connect it to other process model items, by clicking on the start icon and drag and drop the arrow to the to be connected item it.
