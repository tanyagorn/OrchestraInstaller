Creation

To generate a Juno instance press Download Juno instance. This will open following dialog:

 

Download dialog for Juno instance

Please type in a unique name and a password. The password will be needed at starting Juno in the foreign network.

 

The name has to be unique! Otherwise an error will appear!

 

Don't forget the password, otherwise Juno is not connectable.

 

If you press download a Zip file will be downloaded, which must be moved to the foreign network. For the installation of Juno please follow the instructions in Juno Installation chapter.

Once Juno is connected you can see it in the overview as new tab:

Deletion

As you can see the view shows the connection information of the connected Junos, including a state field.

If the instance is connected once it will always be visible in this view, also if it isn't connected any more. This is necessary because in case of a connection loss Orchestra don't know if it is just a temporarily  unexpected error or if it is planed from the user to disconnect the instance.

 

Therefore if you want to delete a Juno instance, please use the Remove Juno instance button. This is the only possibility to delete a Juno from orchestra's registration. Following happens in case of a Juno deletion:

1.Search for Channels, running on the instance (or just declared as Juno channels, if the Channels isn't running or the instance is not reachable)

2.If there are some channels, switch them back to main Orchestra.

3.If Juno is connected, send shutdown command to the instance. This means Juno gets a Shutdown message and closes itself.

4.Delete all information about this instance from Orchestra runtime. If someone starts this instance again it can't connect any more.
