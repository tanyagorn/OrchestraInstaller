Orchestra Monitor Reference


Topic content

The web interface for the runtime environment is named Orchestra monitor. It is used to manage the complete management and administration of the runtime environment in your browser. In addition to the graphical dashboard, role-based administration and events as well as processes, business keys, system parameters and the status of the individual Orchestra nodes can be monitored.

You can access the web interface using one of the major browsers. Using the Orchestra Monitor, you can handle the technical monitoring, the process oriented interface monitoring and the administration.

Supported browsers for Orchestra Monitor

 

Known problems:

- With Internet Explorer 11 problems with the Dashboard can occur!

- With Chrome it is not possible to hide columns!
