Request Configuration


Topic content

 

For each method the user can define corresponding request objects containing information about method request.

 

1.Right click on Method object and select create request option in order to create new request object.

Figure 56 – REST Service Client - Method Request Creation

2.Click on the created Request object to open configuration panel for it.

Figure 57 – REST Service Client - Method Request Configuration

There are four configuration fields related to the Request object.

 

•Request - The description value of the request object.

•Media Type - defining the format of the method request.

•Message Type - defining the internal Message Type to use for the Request.

•Method parameters - defining the request parameters. Headers, query parameters, etc.

Figure 58 – REST Service Client - Method Style Configuration
