HL7 answer messages


Topic content

Description

The generated answer messages can be adjusted in every channel instance. The messages are indicated as text, but can have variables to insert message specific field values.

 

Parameters

The following parameters can be inserted in a HL7 message.

Parameter

Description

$(ACK.CODE)

This will be replaced by the acknowledge code from the configuration

$(ACK.TEXT)

This will be replaced by the description from the configuration

$(ACK.DETAIL)

This will be replaced by the text of the exception (if available)

$(ctrl:counter)

This will be replaced by a distinct identifer

$(dt:<date/time format>)

This will be replaced by the current date or time

$(msg:ID)

This will be replaced by the message identifier of the Orchestra message

$(msg:<name>)

This will be replaced by a mesage property which is included in the Orchestra message
