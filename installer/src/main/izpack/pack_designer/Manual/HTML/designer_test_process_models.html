How to test process models


Topic content

If you want to test a complete process model you have to take a look at your process first. The testing depends on your starting event and the parameters needed by the process model.

The following test process model could be use start event for testing.  

 

1. Drag and drop start event connect to activity in process model.

 With this start event it is very easy and consistent to the other tests

Testing process model which start with message events. 

 

2. Click start button to start process engine.

Testing process model - Engine state

 

3. Right-click in process model would be test. Then, select "Test process model" .

       

Test process model option

 

4. Select prepared test message in CURMSG.Then, click Execute process to test process model.

Test process model dialog

 

5. Then, you can find message flow in each of activity in Process execution.

Log of test process model - Process execution
