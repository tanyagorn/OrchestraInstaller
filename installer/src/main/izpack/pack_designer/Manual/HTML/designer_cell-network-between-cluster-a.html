Cell network between cluster and standalone nodes


Topic content

Cell A is configured as a cluster system. Cell A has to communicate with cell B. The correct configuration for this scenario requires that each node of cell has a connection to the node B.1.

 

Important configuration principle: Each node in a cluster requires at least one connection to a node in the connected cell!
