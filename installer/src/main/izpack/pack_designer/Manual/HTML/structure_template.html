Topic content

Please just use the given styles for your text. Do not change text size or color by hand.

We are working with a separated style repository. This is necessary for the possibility of style changes afterwards.

 

You can create references to all other "Topics".  For example to Message types, or to the IDOC mapping . Therefore drag the intended topic to the right place in your text.

 

To jump to the anchor, set in topic Filepath, choose Write>Insert>Link and select Topic Link. Choose the intended topic and choose instead of "(top)" the anchor in this topic.

 

Screen shots can be done easily with "Screen Capture" under Project > Tools.

 

The project is stored at your local system, if you want to commit your changes to SVN press "Synchronize SVN" under Project > Project.

For updates press "Refresh Project" in the same section.

 

The convention for image names is: <topic id>(accessible in tab "Topic Options")_<image_description>.

 

To add additional files, like Configurations, add the file to folder "baggage" and link it to your topic afterward's via insert >link>file link

Do not create sub folders for resources! An export to WebHelp wouldn't be possible otherwise.
