LPD Listener


Topic content

Description

A LPD Listener is used to receive print jobs.

 

Purpose of object

There are different ways for Orchestra to interpret those print jobs. To do so, it is possible to change the type to DEFAULT or SAPGOF(U).

 

If DEFAULT is chosen, the print job is just written in the CURMSG. SAPGOF(U) are two special print formats from SAP, SAPGOF and SAPGOFU. Those two differ from each other with their charcter set (UTF-16BE or US-ASCII). Orchestra recognizes the format autonomously.

 

Both SAP formats include metainformation which are inserted in the properties of the CURMSG. If you want to know more about SAPGOF, you can find it in this link: http://www.sdn.sap.com/irj/scn/go/portal/prtroot/docs/library/uuid/10dd1e35-a012-2a10-a6a2-c688a2b6793a?QuickLink=index&overridelayout=true&15448997423682 (Despite careful checking, we assume no liability for the content of external links.)
