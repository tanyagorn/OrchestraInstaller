A stored procedures with result of a select


Topic content

Aufruf einer Stored-Procedure, die die Ergebnismenge eines Selects zurückliefert

Definition der Stored-Procedure:

CREATE PROCEDURE [dbo].[selectRecords]
AS
BEGIN
select id, col1, col2 from sample_proc
END

 

Channels-Konfiguration:

Properties:  
RESULT[message, OUTPUT]  
 
Imports:
import java.sql.Connection;
import java.sql.CallableStatement;
import java.sql.ResultSet;
import java.sql.Types;
 
/* ******************************************************************************
* This sample explains the invocation of a stored procedure that  
* returns a result set
*  
*   Step 1) Obtain a database connection from orchestra
*   Step 2) Prepare the invocation of the stored procedure
*   Step 3) Execute the stored procedure
*   Step 4) Create a message from the result
*   Step 5) Close the statement and resultset
*  
* Note: A commit is done automatically
*
********************************************************************************/
 
/*
* Step 1) Obtain a database connection from orchestra
*/
 
Connection     connection = soffico.customerapi.landscape.DatabaseAdapter.getConnection( this.getScenarioIdentifier(), "env_testdatabase" );
 
/*
* Step 2) Prepare the invocation of the stored procedure
*/
 
CallableStatement   statement  = connection.prepareCall( "{call selectRecords()}" );
ResultSet      resSet     = null;
 
try {
/*
  * Step 3) Execute the stored procedure
  */
 
 resSet = statement.executeQuery();
 
/*
  * Step 4) Create a message from the result
  */
 
 RESULT = soffico.customerapi.message.MessageAdapterDatabase.createMessageFromResultSet( resSet, "selectRecords","urn:db","row",false);
}
finally {
 
/*
  * Step 5) Close the statement and resultset
  */
 
if( resSet != null ) {
   resSet.close();
 }
 
if( statement != null ) {
   statement.close();
 }
}
