HCM reader


Topic content

Description

The HCM reader parses HCM formatted messages and creates a list of structured Messages (a messagelist) from it.

Creation

A HCM reader is created, like all other deserializers, by selecting it in the according Stream channel. The referenced XML Schemas must be imported in Orchestra in order to create the data structure.
